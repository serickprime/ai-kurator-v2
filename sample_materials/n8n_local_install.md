# n8n local install

Материал отвечает на вопрос: как установить n8n локально и открыть интерфейс.

## Запуск через npx

Для учебного локального запуска можно использовать:

```bash
npx n8n
```

После старта откройте интерфейс в браузере:

```text
http://localhost:5678
```

Проверка запуска: в терминале нет ошибки, а браузер открывает редактор n8n.

## Запуск через Docker

Если нужен контейнер, используйте проброс порта 5678:

```bash
docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
```

Важно: если порт занят или Docker не запущен, интерфейс на `localhost:5678` не откроется.
